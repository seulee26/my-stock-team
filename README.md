# my-stock-team 📈

**종목 이름만 넣으면, AI 분석가 팀이 알아서 주식 리서치를 해주고 발표용 PPT까지 만들어 주는 도구입니다.**

예를 들어 `삼성전자` 라고 입력하면 —
1. 여러 명의 AI 분석가가 **동시에** 그 종목을 조사하고 (차트·뉴스·재무·위험),
2. 팀장 역할 AI가 그 결과를 하나로 정리하고,
3. 검수 담당 AI가 틀린 곳이 없는지 검사한 뒤,
4. **깔끔한 PPT 파일**로 만들어 줍니다.

> ⚠️ **꼭 읽어주세요:** 이 도구는 **공부·연습용**입니다. "이 주식을 사라/팔아라" 같은 **투자 권유가 아닙니다.** 판단의 재료를 정리해 줄 뿐, 최종 결정은 사람이 합니다.

---

## 🤔 이게 누구한테 필요한가요?

- 주식 공부를 하는데 매번 차트, 뉴스, 재무제표를 따로 찾아보기 번거로운 분
- 종목 정리 자료를 **PPT로 예쁘게** 만들고 싶은 분
- 개발은 잘 모르지만, 시키면 알아서 해주는 AI 도구를 써보고 싶은 분

---

## 📋 시작하기 전에 (3가지 준비물)

아래 3개를 먼저 컴퓨터에 깔아야 합니다. 한 번만 설치하면 됩니다.

### 1) Claude Code (이 도구를 실행하는 프로그램)
이 플러그인은 **Claude Code** 라는 프로그램 안에서 돌아갑니다. 설치 방법은 공식 안내를 따라 주세요 👉 https://docs.claude.com/claude-code

### 2) 파이썬 (Python) — PPT를 만드는 엔진
PPT 파일을 만드는 데 필요합니다. 👉 https://www.python.org/downloads/ 에서 내려받아 설치하세요.
> 설치할 때 화면 아래에 있는 **"Add Python to PATH"** 체크박스를 꼭 켜주세요. (안 켜면 컴퓨터가 파이썬을 못 찾습니다.)

### 3) DART 키 — 한국 기업 재무 데이터를 가져오는 열쇠 (무료)
한국 종목의 재무제표는 금융감독원 **DART** 라는 공공 사이트에서 가져옵니다. 여기서 **무료 열쇠(키)** 를 발급받아야 합니다.
1. 👉 https://opendart.fss.or.kr 접속 → 회원가입
2. 로그인 후 **[인증키 신청/관리]** 에서 키 발급 (긴 영문+숫자 문자열을 줍니다)
3. 이 키를 잠시 메모장에 복사해 두세요. (아래 설치 4단계에서 씁니다)

> 미국 종목만 볼 거라면 DART 키는 없어도 됩니다. (단, 한국 종목의 재무·위험 분석은 빠집니다.)

## 🚀 설치 방법 (복사해서 붙여넣기만 하면 됩니다)

Claude Code를 켠 다음, 입력창에 아래 명령을 **한 줄씩** 붙여넣고 엔터를 누르세요.

**1단계 — 이 도구를 Claude Code에 등록하기**
```
/plugin marketplace add seulee26/my-stock-team
```

**2단계 — 도구 설치하기**
```
/plugin install my-stock-team@my-stock-team
```

**3단계 — PPT 엔진에 필요한 부품 설치하기**
컴퓨터의 "터미널"(명령 입력 창)에 아래를 붙여넣으세요. Claude Code 입력창이라면 맨 앞에 `!` 를 붙이면 됩니다.
```
pip install python-pptx FinanceDataReader pykrx OpenDartReader python-dotenv
```
> `pip` 이 안 먹히면 `pip` 대신 `py -m pip` 또는 `python -m pip` 으로 바꿔서 다시 해보세요.

**4단계 — DART 키 넣어두기 (한국 종목 재무 분석용)**
작업하는 폴더 안에 `.env` 라는 이름의 메모장 파일을 하나 만들고, 그 안에 아래 한 줄을 적습니다. `발급받은_키` 자리에 위에서 복사해둔 진짜 키를 붙여넣으세요.
```
DART_KEY=발급받은_키
```
> 🔒 이 `.env` 파일은 **나만 보는 비밀 파일**입니다. 절대 다른 사람에게 보내거나 인터넷에 올리지 마세요. (이 도구는 자동으로 이 파일이 업로드되지 않게 막아둡니다.)

## ▶️ 사용 방법

설치가 끝났으면, Claude Code 입력창에 이렇게만 적으면 됩니다.

**종목 분석 + PPT 만들기 (전체 자동)**
```
/my-stock-team:analyze 삼성전자
```
종목 이름은 한국 주식(예: `SK하이닉스`), 미국 주식(예: `애플` 또는 `AAPL`) 다 됩니다.

**이미 만든 리포트 검사만 다시 하기**
```
/my-stock-team:verify 삼성전자
```

### 결과물은 어디에 생기나요?
작업 폴더 안의 **`reports`** 라는 폴더에 두 개가 생깁니다.
- `삼성전자.md` — 글로 정리된 리서치 (메모장으로 열림)
- `삼성전자.pptx` — **발표용 PPT 파일** (파워포인트로 열림)

## 👥 AI 분석가 팀은 누가 있나요?

| 분석가 | 하는 일 | 쉽게 말하면 |
|---|---|---|
| 기술적 분석가 | 주가 추세·거래량·돌파 | "차트가 오르는 모양인가?" |
| 뉴스·심리 분석가 | 뉴스·이슈·시장 분위기 | "요즘 분위기가 좋은가?" |
| 리스크 분석가 | 급락·과열·변동성 점검 | "지금 위험하진 않은가?" |
| 펀더멘털 분석가 | 회사 재무·실적 | "회사 체력은 튼튼한가?" |
| 검증 분석가 | 리포트 오류 검사 | "정리된 내용에 틀린 곳 없나?" |

각 분석가는 **출처와 날짜가 붙은 숫자만** 사용하고, 못 구한 정보는 솔직하게 "확인 불가"라고 표시합니다.

## ❓ 자주 막히는 곳

- **"명령어가 안 먹혀요"** → 1·2단계(`/plugin ...`)는 Claude Code 입력창에, 3단계(`pip ...`)는 터미널에 넣는 거예요. 헷갈리면 그냥 "이거 어디다 넣어?" 라고 Claude Code에 물어보세요.
- **"python을 찾을 수 없다고 나와요"** → 파이썬 설치할 때 "Add Python to PATH" 체크를 안 한 경우예요. 파이썬을 지우고 다시 설치하면서 그 체크박스를 켜주세요.
- **"한국 종목 재무가 안 나와요"** → 4단계 DART 키가 빠졌거나 잘못 들어간 경우예요. `.env` 파일의 키를 다시 확인하세요.
- **"PPT 글자가 깨져요"** → 한글 폰트 **맑은 고딕**이 필요합니다. 윈도우는 기본으로 있고, 맥/리눅스라면 맑은 고딕(또는 다른 한글 폰트)을 설치하면 됩니다.
